# Orange-UI特供公共图标库（暂时只有vue）

## Installation

```bash
$ npm install orange-ui-icon
```

## Basic usage

```html
<script>
import { Add } from 'orange-ui-icon'
</script>
......
<template>
  <Add></Add>
  <Add style="font-size: 24px;color: #ec9b64"></Add>
  <Add :style="{ 'font-size': '24px', color: '#ec9b64' }"></Add>
</template>
```
