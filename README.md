# Orange-UI特供公共图标库（暂时只有vue）

## Installation

```bash
$ npm install orange-ui-icon
```

## Basic usage

```html
<script>
import Jia from 'orange-ui-icon/icon/Jia'
</script>
......
<template>
  <Jia></Jia>
</template>
```
